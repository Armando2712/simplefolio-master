Emerson Electric Co (Academic Project)

At Emerson, we were tasked with developing a performance measurement system for two plants located in Mexico. We were provided with various databases where we had to employ different methodologies such as principal component analysis, clustering analysis, and multivariate factor analysis, as well as data analysis and visualization tools like R, Tableau, Power BI, Data Studio, among others, to establish various Key Performance Indicators (KPIs) that could measure the performance of Emerson's plants. This was well-received by Emerson, and there was even a discussion about the potential implementation of this new key performance indicator.
